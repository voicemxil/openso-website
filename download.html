Download the OpenSO launcher for your platform — it installs the game, keeps it up to date, and helps you fetch the original The Sims Online files. OpenSO never redistributes EA assets.

First time? How to get started:

  1. Create your free account.
  2. Download & run the OpenSO launcher for your platform above.
  3. Install the game in the launcher and then hit play — it sets the game up for you.

Supported platforms

The OpenSO launcher is native on all three major platforms.

  • Windows — 10/11, 64-bit.
  • macOS — Apple Silicon (arm64) & Intel (x64).
  • Linux — 64-bit (x64), portable .zip.

Game broken, or an update failed?

Just run the install in the OpenSO launcher again — it checks your install and repairs anything missing or corrupted without touching your saves. If the game still won't start, download the launcher fresh from above and let it reinstall.
