OpenSO is a modernized, community-run revival of **The Sims Online**, built on the open-source FreeSO
engine. The official servers are gone — so we're building a polished, fair, actively-run place to play,
aimed at welcoming new players, not just preserving the old game.

## What makes it different

- **Never pay-to-win.** No paid items, no donations for perks, no advantages for money. Everyone plays on
  the same field.
- **Modernized.** Updated graphics and performance, with native Windows, Linux, and macOS clients.
- **Open source.** Public code and public builds — you can always verify what you're running.
- **Welcoming & moderated.** Active moderation, a friendly community, and a steady cadence of events.

## How to follow along

This is where we'll post updates — new features, events, server news, and release notes. The launcher
reads this same feed, so you'll see the latest right when you sit down to play.

Want in early? **Create an account**, grab the client from the Download page, and come say hi.
